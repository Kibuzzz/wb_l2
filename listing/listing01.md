Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Будет выведено [77, 78, 79] потому что в слайс [1:4] включит в себя элементы с индексами от 1 до 3 (4 не включительно)
[76, 77, 78, 79, 80] 
  0,  1,  2,  3,  4 индексы
```
